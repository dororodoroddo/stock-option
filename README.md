# Stock option
커스텀화 가능한 주식 자료수집 플랫폼

## 사이트 목적
- 어닝 등 일정 관리 *
- 특정 주식에 대한 여러 관련 키워드 동시 검색 *
- 투자전략 메모 및 과거 전략 분석 및 개선 유도
- 비슷한 유형의 주식 한번에 표시(그룹화), 어닝 등 메모를 통한 비교 (like 애플)
- 가격 확인
- 가상 투자
-
